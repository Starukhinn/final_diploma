Открытые геоданные OpenStreetMap для вашей ГИС
==============================================

Источник: OpenStreetMap (https://osm.org) - открытая база геоданных и сообщество редакторов, поддерживающих её в актуальном состоянии.

Получить свежие данные OpenStreetMap в удобной форме можно с помощью сервиса: https://data.nextgis.com

Источник кодов ОКТМО в слоях административных границ и населенных пунктов - https://rosstat.gov.ru/opendata/7708234640-oktmo 

Каждая выгрузка содержит:

* слои данных (папка data)
* иконки для условных обозначений (папка svg)
* готовый проект для вашей ГИС
